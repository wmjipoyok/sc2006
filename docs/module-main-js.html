Description:

(inner) getData()

Description:

  • The function retrieves car park details from the URA API (ura.gov.sg) and adds markers to a map based on the coordinates provided in the API response. The API returns the car park coordinates in Singapore SVY21 format so svy21.js library is used to convert the coordinates to latitude and longitude coordinates and so that we can display the car park locations in Google Maps.
